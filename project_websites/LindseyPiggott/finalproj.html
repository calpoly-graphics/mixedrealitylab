Mountains with Flowing River

Lindsey Piggott - CSC 471 - Spring 2018 - Christian Eckhardt

Project Description

The goal of the project was to create a realistic looking landscape with mountains and a river flowing beneath. To create a more realistic look, the mountains were given a realistic texture as well as lighting. In addition, the water is animated and reflects the skybox.
