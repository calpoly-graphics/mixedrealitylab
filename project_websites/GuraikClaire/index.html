A sub-scene of Bikini Bottom

Main Neighborhood of Bikini Bottom

For my Final Project, I made a scene of Bikini Bottom. My scene does not include the entire town, but rather some of the key components of the town. Components of Bikini Bottom to be included:

  • The main neighborhood, which includes:
    • Spongebob's house
    • Squidward's house
    • Patrick's house
  • The Krusty Krab
  • Jellyfish Fields

My project also includes characters from the show placed among the scene! Characters included:

  • Spongebob Squarepants
  • Patrick Star
  • Squidward Tentacles
  • Jellyfish

The unique thing about my project is that I did not use any objects or shapes except for:

  • Sphere
  • Cube
  • Cylinder
  • Cone
  • Billboard
Meaning that everything created in this scene was created by using transformations on these shapes and combining them to create what I wanted.

Technical Implications of this project:

  • Hierarchical Modeling
  • Diffuse, Ambient, and Specular Lighting
  • Textures
  • Skybox
  • Animated Billboards
  • Height Mapping
  • Instance Rendering
  • Frame Buffers

Jellyfish Fields

Features and Instructions

The main feature of the entire scene is that by pressing 'O', the scene will begin to have an underwater effect, or go back to normal if underwater effect is occurring.

Main Neighborhood

The only feature of the neighborhood is that by pressing 'P', Patrick will come out of his rock, or he will go back in if already out.

Jellyfish Fields

The only feature of Jellyfish Fields is the animated billboards causing the Jellyfish to have a 'swimming' effect.

Controls

  • W - Move Forward
  • S - Move Backward
  • A - Turn Left
  • D - Turn Right
  • O - Toggle Underwater Effect
  • P - Toggle Patrick Animation
